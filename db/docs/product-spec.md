# Loomap Product Spec

## Vision
Loomap helps people quickly find trustworthy public bathrooms nearby.

## Core principles
- Fast in urgent moments
- Accessibility-first
- Anonymous browsing
- Community-updated
- Trust made visible

## Sprint roadmap
1. Product spec, flows, schema, design system
2. Map view, list view, geolocation, search, filters
3. Detail page, ratings, trust badges
4. Add, edit, report flows
5. Admin moderation queue, duplicate detection, edit review
6. PWA, offline fallback, performance tuning
7. Reputation, contributor profiles, optional Web3
